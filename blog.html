Advanced DocuSign Integration for Salesforce

Client Project | 2024

I led an advanced Salesforce DocuSign implementation that automated legal document generation, pre-filled dynamic fields using Apex, and sent envelopes directly from Salesforce. The automation included event-based triggers and tracking statuses within Salesforce. Custom LWC components were used to track the document lifecycle, and webhook-based callbacks were integrated with Apex REST for real-time syncing.

Adobe Sign Integration for Salesforce

Client Project | 2023

This implementation supported document approvals and user-friendly UI for sending sign requests from record pages. I built custom screen flows and Apex classes for handling Adobe Sign APIs, implemented OAuth for secure access, and configured Named Credentials. The solution allowed tracking of document status and centralized logging using Platform Events.
